Project Overview

This project focuses on creating a smooth online shopping experience for a bakery brand. The interface emphasizes visual appeal, intuitive browsing, and a frictionless checkout flow for users on desktop and mobile.

Key Features

  • Responsive product catalog with visually engaging product cards.
  • Shopping cart flow with quantity updates and order summary handling.
  • Clean UI system built to support brand consistency and easy navigation.
